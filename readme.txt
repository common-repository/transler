== Description ==

Осуществляет транслитерацию кириллицы в название файлов и слагах записей.

== Installation ==

1. Загрузите `transler` в папку `/wp-content/plugins/`.
2. Активируйте плагин в разделе 'Плагины' в меню WordPress.
